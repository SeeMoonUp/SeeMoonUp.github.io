maven+jetty+idea+jrebel 实现项目热部署

发表于 2017-12-19

开发环境

开发过程中：
修改java或者工程文件 需要重新对工程进行build deploy耗时太久 到无法忍受

如何实现一次部署之后，后面的修改全部热部署呢？

  1. 使用jrebel实现
  2. web容器使用的是jetty
  3. 项目管理使用的maven
  4. ide使用的是idea
