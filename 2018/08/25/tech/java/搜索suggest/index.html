搜索suggest

发表于 2018-08-25 | 分类于 fe ， plugins | 阅读次数:

解决的问题：

  1. 因为select下拉选项太多需要做搜索建议
  2. 提高页面加载速度
  3. 实现一次之后 后续的接入只需修改js和枚举

主要思路：

  1. 使用动态sql 拼装需要的参数
  2. 使用枚举来动态的加载表
  3. 在页面中传入需要的枚举值 和 需要like的字段

引入插件magicsuggest

在页面中加上相应的区域 可以为input或者div

在js中对上面的id进行绑定magicsuggest事件

设置枚举类对应请求参数

请求后台sql mybatis
