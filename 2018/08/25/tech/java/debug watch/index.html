debug模式下调试工具的使用
主要说一个点 那就是watch这个小功能的使用
从前端工具（chrome）服务端工具（ieda）分别来介绍

chrome浏览器F12

  • 在chrome浏览器中F12可以打开调试工具，
  • 在source中可以点击行数 即可加上断点，
  • 在旁边的watch一栏中点击“+”号即可在输入框内使用当前方法体内的所有变量
  • 加快调试的进度

idea编辑器

接下来介绍idea中的调试工具

  • 在debug模式下启动项目
  • 在运行流程中加上断点
  • 运行程序使程序进入断点
  • 进入console使用watches即可获取当前断点内的全部可用变量
  • 运行变量 检查变量 加快调试
